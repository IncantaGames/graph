Detailed Description

template<typename V, typename... Ts>
class graph::PartiteGraph< V, Ts >

An undirected graph that support multiple named partitions with different types. V is used to define the type for the partition names while Ts is used to define the types for each partition.

Template Parameters
Vtype for partition names
Tstypes for the partition values
